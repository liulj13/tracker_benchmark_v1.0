This code implements the tracking algorithm in the paper: 

"Tracking with Distribution Fields" 
L. Sevilla-Lara and E. Learned-Miller 
To appear at CVPR 2012. 

The project website is: 

http://people.cs.umass.edu/~lsevilla/trackingDF.html

-------------------------------------------------------------------------------------------

Instructions: 

1) In MATLAB, go to the directory where the code is. 
2) Run "trackingDemo.m" 

This will compute the track for the sequence "cokecan", stored in the folder called "data". 
This will also display the results on your screen and create a avi file with these results.
